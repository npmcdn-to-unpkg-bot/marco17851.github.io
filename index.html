A-Maze-ing Project

The A-Maze-ing Project was the final project for my Software Design and Implementation course at Dartmouth. We were to find a way for avatars inside the maze to find each other, without knowing anything about the maze, except where the other avatars were located. As a result, my partners and I created our own maze depending on where the avatars were going and where they were finding walls. This project involved setting up a connection to a server, being able to make each avatar talk to the server one by one, work with different processes, implement an algorithm to find the other avatars, implement a way to create our own maze, and ultimately, show the solution to each maze with simple graphics.

Rankr

Rankr was my senior project at Dartmouth. It is a movie ranking website whose users determine movies’ Elo ratings by viewing two movies side-by-side and choosing which is better. The website lists movies ranked by their Elo ratings and can be filtered by genre. It also compiles a user’s personal top 10 list, determined from comparisons the user has made. It also provides movie recommendation that function by 1) matching each user with another user that has a similar (but not too similar) personal top 10 and 2) providing movie recommendations based on movie genres in the personal top 10.
